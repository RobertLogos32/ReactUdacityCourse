# MyReads Project

A React application that allows users to organize books into three shelves: **Currently Reading**, **Want to Read**, and **Read**. Users can search for new books and add them to their shelves, move books between shelves, or remove them. The app communicates with a Udacity backend API and uses React Router for client-side navigation.

## Installation and Setup

To download and start the project from GitHub:

1. `git clone https://github.com/RobertLogos32/ReactUdacityCourse.git`
2. `cd starter`
3. `npm install`
4. `npm start`
